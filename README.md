# 📰 الأخبار اليوم - موقع إخباري احترافي

موقع إخباري عربي شامل وحديث يعرض أحدث الأخبار المحلية والعالمية.

## ✨ المميزات
- 🔴 شريط أخبار عاجلة متحرك
- 🏠 قسم بطل (Hero Section) لأبرز الأخبار
- 🗂️ تصفية الأخبار حسب التصنيف
- 🔍 بحث فوري في الأخبار
- 🌙 وضع ليلي / نهاري
- 📱 تصميم متجاوب لجميع الشاشات
- 📊 إحصائيات الموقع المتحركة
- 🔔 نشرة بريدية
- #️⃣ وسوم شائعة
- 📤 مشاركة الأخبار

## 🚀 نشر على GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. ابحث عن **Pages**
3. اختر **main** branch وحفظ

## 🛠️ التقنيات
- HTML5 + CSS3 (Variables, Grid, Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Cairo)
